# AI Usage Log

## Overview

During the development of this assignment, AI-assisted tooling (ChatGPT) was used as a productivity aid to accelerate development, troubleshoot issues, and review implementation approaches.

The final implementation, testing, debugging, and validation were performed manually by the developer.

---

## Areas Where AI Assistance Was Used

### 1. Solution Design

AI was used to discuss possible approaches for implementing the uptime monitoring system, including:

* Frontend and backend separation
* API structure
* Database design
* Scheduling mechanisms for periodic health checks

These discussions helped evaluate different implementation options before development began.

---

### 2. Backend Development

AI assistance was used to review:

* Django project structure
* Django REST Framework patterns
* Model relationships
* API endpoint implementation
* Health-check scheduling logic

All generated suggestions were reviewed and adapted to fit the final solution.

---

### 3. Frontend Development

AI was used to improve the presentation layer by:

* Reviewing React component structure
* Improving dashboard layout
* Enhancing table rendering
* Refining the overall user experience

Several UI suggestions were modified before integration.

---

### 4. Dockerization

AI assistance was used while containerizing the application:

* Dockerfile review
* Docker Compose troubleshooting
* Container networking discussions
* Local deployment validation

---

### 5. Documentation

AI was used to review and improve:

* Project documentation
* Setup instructions
* Deployment sketch
* Verification procedures
